# BIM Constructions

Welcome to the BIM Constructions project repository!

## Overview
This is a modern web application for **BIM Constructions**, delivering high-performance and scalable digital solutions for professional engineering and construction services.

## Technologies
- **Next.js 16** (App Router)
- **React 19**
- **i18n** (Internationalization for Spanish and English)
- **Feature-Sliced Design (FSD)** architecture for scalable, modular codebase organization
- **CSS Modules** for scoped styling

## Setup & Running

```bash
pnpm install
pnpm run dev
```
